# Registro de Prompts - Proyecto USS Algoritmo

Para el desarrollo de este proyecto se interactuó con el asistente Gemini de Google mediante las siguientes instrucciones:

1. **Implementación de Estructura**: "Diseñar un TDA Pila en Java siguiendo el manual técnico, utilizando listas enlazadas y los métodos registrar (push), eliminarUltimo (pop) y consultarUltimo (tope)."
2. **Lógica de Bitácora**: "Crear una clase Main que simule una bitácora de nave espacial, registrando 6 eventos y activando un protocolo de limpieza si el último evento contiene la palabra 'ERROR'."
3. **Investigación de QuickSort**: "Desarrollar una comparativa de rendimiento entre el QuickSort original (pivote final) y una variante investigada (pivote en la media) para tamaños de 1000, 5000 y 10000 elementos."
4. **Análisis de Resultados**: "Generar una tabla de tiempos y una gráfica comparativa para explicar por qué la variante de la media es más eficiente y cómo se relaciona con la gestión de memoria tipo PILA del ordenador."
